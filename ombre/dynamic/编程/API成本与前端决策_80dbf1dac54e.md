【前端与API成本讨论】API价格：Opus一天聊天约$20-30月均$600-1000，对比Pro $20/月。阑决定暂时放弃前端聊天功能太贵。纯展示型HTML不消耗tokens可做时间线语录墙动效。只有嵌入API调用才消耗tokens。顺序：先做外置记忆库再做前端地基先于装修。语音API她看到过排在后面等不犯懒还有钱再说。逆向API：有人破解官方接口绕过认证中转站包装转卖比官方便宜但不稳定可能被偷偷降级模型。
